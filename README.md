## LightSetup
Basic setup for small projects

# Clone repository
$git clone https://github.com/Wojtek534/LightSetup.git

# Install dependencies
$npm install

# Run app
$npm start

###

## Dist
# Get bundle
$webpack

# Crate dist
$gulp dist

# NPM Save config (--save)
$npm config set save=true

## Share
$npm start

# Add terminal
$npm run lt
